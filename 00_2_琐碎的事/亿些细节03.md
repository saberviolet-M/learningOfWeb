### 2.清空数组的操作

- 1、重新赋值为[]

  ```js
  arr = [];
  ```

- 2、设置数组长度为0

  ```js
  arr.length = 0;
  ```

- 3、把数组中的所有数据删除

  ```js
  arr.splice(0,arr.length);
  ```

### 3.Js赋值样式时需要添加单位

```js
let x = [变量值];
element.style.width = x + 'px';
```

###  4.querySelectorAll和children的区别

- querySelectorAll：获取的元素集合是静态集合——拿到数据之后就不管之后的变化了
- children：获取到的元素是动态集合——拿到数据之后还会动态更新数据
